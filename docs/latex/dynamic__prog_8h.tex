Resolve uma versão do problema Knapsack com programação dinâmica. 

Com programação dinâmica, o algoritmo guarda numa tabela as soluções parciais ótimas de etapas anteriores, para calcular soluções ótimas de etapas posteriores. Primeiro,completamos os casos base da tabela e depois, para cada “\+Pallet”, verificamos se é melhor colocá-\/la ou não no “\+Truck.
