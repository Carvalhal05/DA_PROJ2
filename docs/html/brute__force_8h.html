Resolve uma versão do problema Knapsack com brute force. Com este método, o algoritmo testa todas as combinações possíveis de “Pallets”, verificando para cada uma a restrição de peso e o seu valor. Depois de avaliar todas as combinações, escolhe a melhor opção.

Parameters
valuesvetor com o valor de cada "Pallet"
weightsvetor com o peso de cada "Pallet"
nnúmero de "Pallets"
maxWeightcapacidade máxima do "Truck"
usedItemsvetor que indica quais "Pallets" estão no "Truck"
Note
Complexidade: O(2^n), onde n é o número de "Pallets".
